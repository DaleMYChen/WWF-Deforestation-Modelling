WCF Data Info

Data Description

Each file contains high-resolution (10m) WCF data in GeoTIFF format. Values range from 0-100, representing percentage cover. No-data values are set to 0.

Spatial Extent

The Australian continent is divided using a 4x4 grid defined in Australian Albers Equal area projection. Each tile is identified with a number starting at the centre of the grid. This allows users to download just the areas of interest.

Temporal Extent

WCF data is available for each of the years 2022 and 2023.
